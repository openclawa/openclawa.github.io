工作原理

OpenClaw 的核心是运行在您本地机器上的 Web Socket 网关。它充当所有客户端、工具和事件的统一控制平面。

  • 网关控制平面 (ws://127.0.0.1:18789)
  • Pi Agent (RPC) 运行时
  • Tailscale Serve/Funnel 远程访问
  • Docker 沙箱隔离 (安全运行)
